# Merge Three Sorted Arrays

Merge two ascending arrays and one descending array into a single sorted array by ascending. No `sort()` function used.

## How to setup dependency and execute code and unit test

make sure you are at merge-interview file
```bash
    cd merge-interview
```

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Tests
```bash
npm test
```

### 3. Run Demo
```bash
npm run dev
```
